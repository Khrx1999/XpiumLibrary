ยินดีต้อนรับสู่เอกสารของ Robot Framework XLibrary!
====================================================

XLibrary เป็นไลบรารีที่ทรงพลังสำหรับ Robot Framework ที่ให้คุณสมบัติการทำงานอัตโนมัติเพิ่มเติมและการบูรณาการกับระบบภายนอก.

แนะนำ
--------
หน้านี้จะแนะนำคุณเกี่ยวกับไลบรารีและคุณสมบัติหลักที่มีอยู่.

การติดตั้ง
------------
บทนี้จะอธิบายวิธีการติดตั้งและตั้งค่า XLibrary.

การใช้งาน
---------
ตัวอย่างและสคริปต์ตัวอย่างในการใช้งาน XLibrary ในโปรเจ็กต์ของคุณ.

API Reference
--------------
API เต็มรูปแบบของ XLibrary, พร้อมรายละเอียดของ keywords และความสามารถต่างๆ.

การมีส่วนร่วม
---------------
คำแนะนำสำหรับการมีส่วนร่วมในการพัฒนา XLibrary.

บันทึกการเปลี่ยนแปลง
-------------------
ประวัติของการเปลี่ยนแปลงและการอัปเดตที่สำคัญในเวอร์ชันต่างๆ.
